# wikinear
<h2>Building</h2>

1. Create a .env file in the root dir
Get api key from https://positionstack.com/product
```cmd
POSITIONSTACK_API_KEY=own_api_key
```
2. Get all dependencies
```cmd
flutter pub get
```
3. Then build the app
android:
```cmd
flutter build apk --release
```
ios use xcode
